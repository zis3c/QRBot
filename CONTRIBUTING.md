# Contributing to QRBot

Thank you for your interest in contributing to QRBot! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** locally with a succinct but descriptive name.
4.  **Make your changes** to the code.
5.  **Test your changes** to ensure everything works as expected.
6.  **Commit your changes** with a clear commit message.
7.  **Push changes** to your fork.
8.  **Open a Pull Request** in our repository.

## Development Setup

1.  Ensure you have Python 3.8+ installed.
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3.  Set up environment variables (see README.md).
4.  Run the bot:
    ```bash
    python bot.py
    ```

## Code Style

- Please follow standard Python PEP 8 style guides.
- Keep code clean and readable.
- Add comments where necessary.

## Reporting Issues

If you find a bug or have a feature request, please open an issue in the Issue Tracker.
